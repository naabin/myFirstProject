General Fomulas

  • $\\\displaystyle\large\frac{d}{dx} c = 0 $
  • $\\\displaystyle\large\frac{d}{dx}\Big[f(x) + g(s)\Big] = f'(x) + g'(x)$
  • $\\\displaystyle\large\frac{d}{dx}\Big[f(x)g(x)\Big] = f'(x)g(x) + f(x)g'(x)$
  • $\\\displaystyle\large\frac{d}{dx}x^{n} = nx^{n-1}\Big[\text{For real numbers n}\Big]$
  • $\\\displaystyle\large\large\frac{d}{dx}cf(x) = cf'(x)$
  • $\\\displaystyle\large\frac{d}{dx}\big[f(x) - g(x)\Big] = f'(x) - g'(x)$
  • $\\\displaystyle\large\frac{d}{dx}\Bigg[\frac{f(x)}{g(x)}\Bigg] = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^{2}}$

Trignometric Functions

  • $\\\displaystyle\large\frac{d}{dx}\sin x = \cos x $
  • $\\\displaystyle\large\frac{d}{dx}\tan x = \sec^2 x $
  • $\\\displaystyle\large\frac{d}{dx} \sec x = \sec x\ \tan x $
  • $\\\displaystyle\large\frac{d}{dx} \cos x = -\sin x $
  • $\\\displaystyle\large\frac{d}{dx} \cot x =-\csc^2 x $
  • $\\\displaystyle\large\frac{d}{dx} \csc x = -\csc x\ \cot x $

Inverse Trignometric Functions

  • $\\\displaystyle\large\frac{d}{dx}\arcsin^{-1} x = \frac{1}{\sqrt{1 - x^{2}}} $
  • $\\\displaystyle\large\frac{d}{dx} \arccos^{-1} x = - \frac{1}{\sqrt{1-x^{2}}}$
  • $\\\displaystyle\large\frac{d}{dx} \sec^{-1} x = \frac{1}{|x| \sqrt{x^{2}-1}} $
  • $\\\displaystyle\large\frac{d}{dx} \cot^{-1} x = - \frac{1}{1 + x^{2}}$
  • $\\\displaystyle\large\frac{d}{dx} \csc^{-1} x = - \frac{1}{|x|\sqrt{x^{2}-1}} $

Exponential and Logarithmic Functions

  • $\\\displaystyle\large\frac{d}{dx} e^{x} = e^{x} $
  • $\\\displaystyle\large\frac{d}{dx} \ln|x| = \frac{1}{x}$
  • $\\\displaystyle\large\frac{d}{dx} b^{x} = b^{x} \ln b $
  • $\\\displaystyle\large\frac{d}{dx} log_b x = \frac{1}{x\ \ln b}$

Hyperbolic Functions

  • $ \\\displaystyle\large\frac{d}{dx} \sinh x = \cosh x $
  • $ \\\displaystyle\large\frac{d}{dx} \tanh x = sech^{2} x $
  • $ \\\displaystyle\large\frac{d}{dx} sech x = - sech x \ tanh x$
  • $ \\\displaystyle\large\frac{d}{dx} \cosh x = \sinh x $
  • $ \\\displaystyle\large\frac{d}{dx} coth x = -csch^{2} x $
  • $ \\\displaystyle\large\frac{d}{dx} csch x = -{csch x} \ {coth x} $

Inverse Hyperbolic Functions

  • $\\\displaystyle\large\frac{d}{dx} \sinh^{-1}x = \frac{1}{\sqrt{x^{2}+ 1}}$
  • $\\\displaystyle\large\frac{d}{dx}\tanh^{-1}x = \frac{1}{1-x^{2}}\Big(|x| < 1\Big)$
  • $\\\displaystyle\large\frac{d}{dx} sech^{-1}x = - \frac{1}{x\sqrt{1 - x^{2}}} \Big(0 < x < 1\Big)$
  • $\\\displaystyle\large\frac{d}{dx} \cosh^{-1}x = \frac{1}{\sqrt{x^{2}-1}}\Big(x > 1\Big)$
  • $\\\displaystyle\large\frac{d}{dx} coth^{-1} x = \frac{1}{1-x^{2}}\Big(|x| > 1\Big)$
  • $\\\displaystyle\large\frac{d}{dx} csch^{-1}x = - \frac{1}{|x|\sqrt{1 + x^{2}}}\Big(x \ne 0\Big)$
