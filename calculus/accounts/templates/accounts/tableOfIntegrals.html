Basic Integrals

  • $\displaystyle \large\int u^{x} du = \frac{x^{x+1}}{x+1}+ C, x\ne -1 $
  • $\displaystyle \large\int \frac{dx}{x} = \ln|x| + C $
  • $\displaystyle \large\int e^{x}dx = e^{x} + C $
  • $\displaystyle \large\int a^{x} du = \frac{a^x}{\ln x} + C $
  • $\displaystyle \large\int \sin x \ dx = -\cos x + C$
  • $\displaystyle \large\int \cos x \ dx = \sin x + C$
  • $\displaystyle \large\int sec^{2}x \ dx = \tan x + C $
  • $\displaystyle \large\int csc^{2} x \ dx = = - cot x + C $
  • $\displaystyle \large\int sec x \ tan x \ dx = sec x + C$
  • $\displaystyle \large\int csc x \ cot x \ dx = - csc x + C$
  • $\displaystyle \large\int tan x \ dx = \ln|sec x| + C$
  • $\displaystyle \large\int cot x \ dx = \ln|sin x| + C $
  • $\displaystyle \large\int sec x \ dx = \ln|sec x + tan x|+ C$
  • $\displaystyle \large\int csc x \ dx = \ln|csc x - cot x |+ C$
  • $\displaystyle \large\int \frac{dx}{\sqrt{a^{2}-x^{2}}} = \sin^{1}\frac{x}{a} + C$
  • $\displaystyle \large\int \frac{dx}{a^{2} + x^{2}} = \frac{1}{a}tan^{-1}\frac{x}{a} +C$
  • $\displaystyle \large\int \frac{dx}{x\sqrt{x^{2}- a^{2}}} = \frac{1}{a}sec^{-1}\frac{x}{a} + C$

Trignometric Integrals

  • $\displaystyle \large\int sin^{2} x \ dx = \frac{1}{2}x - \frac{1}{4}sin2x +C $
  • $\displaystyle \large\int cos^{2} x \ dx = \frac{1}{2}x + \frac{1}{4}sin2x + C$
  • $\displaystyle \large\int tan^2{x} \ dx = tan x - x + C$
  • $\displaystyle \large\int cot^2{x} \ dx = -cot x - x + C $
  • $\displaystyle \large\int sin^{3} x \ dx = - \frac{1}{3}\Big(2 + sin^{2} x\Big)cos x + C$
  • $\displaystyle \large\int cos^{3} x \ dx = \frac{1}{3}\Big(2 + sin^2 x\Big)sin x + C$

  • $\displaystyle \large\int csc^{3} x \ dx = - \frac{1}{2} csc x \ cot x + \frac{1}{2}ln|csc x - cot x| +C $
  • $\displaystyle \large\int sin^{n} x \ dx = - \ frac{1}{n}sin^{n-1} x \ cos x + \frac{n-1}{n}\displaystyle \int sin^{n-2} dx + C $
  • $\displaystyle \large\int cos^{n} x \ dx = \frac{1}{n}cos^{n-1} x \ sin x + \frac{n-1}{n}\displaystyle \int cos^{n-2} dx + C $
  • $\displaystyle \large\int tan^{n} x \ dx = \frac{1}{n-1}tan^{n-1} x - \displaystyle \int tan^{n-2} x dx + C $
  • $\displaystyle \large\int cot^{n} x \ dx = \frac{-1}{n-1}cot^{n-1} x - \displaystyle \int cot^{n-2} dx + C $
  • $\displaystyle \large\int sec^{n} x \ dx = \frac{1}{n-1}tan x \ sec^{n-2} x + \frac{n-2}{n-1}\displaystyle \int csc^{n-2} dx + C$
  • $\displaystyle \large\int csc^{n} x \ dx = \frac{-1}{n-1}cot x \ csc^{n-2} x + \frac{n-2}{n-1}\displaystyle \int csc^{n-2} dx + C $
  • $\displaystyle \large\int sin \ ax \ sin \ bx \ dx = \frac{sin\big(a - b\big)x}{2\big(a-b\big)} - \frac{sin\big(a+b\big)x}{2\big(a+b\big)} + c$
  • $\displaystyle \large\int cos \ ax \ cos \ bs \ dx = \frac{sin\big(a-b\big)x}{2\big(a-b\big)} + \frac{sin\big(a+ b\big)x}{2\big(a+b\big)} + C $
  • $\displaystyle \large\int sin \ ax cos \ bx \ dx = -\frac{cos\big(a - b\big)x}{2\big(a - b\big)} - \frac{cos\big(a + b\big)x}{2\big(a + b\big)} + C$
  • $\displaystyle \large\int x \ sin x \ dx = sin x - x \ cos x + C$
  • $\displaystyle \large\int x \ cos x \ dx = cos x + sin x + C$
  • $\displaystyle \large\int x^{n} sin x \ dx = -x^{n} cos x + n\displaystyle \int x^{n-1} \ cos x \ dx + C$
  • $\displaystyle \large\int x^{n} cos x \ dx = x^{n} sin x - n \displaystyle \int x^{n-1} \ sin x \ dx + C$
  • $\displaystyle \large\int sin^{n} x \ cos^{m} x \ dx = -\frac{sin^{n-1} x\ cos^{m + 1} x }{n + m} +\frac{n-1}{n + m}\displaystyle\int sin^{n-2}x \ cos^{m}x \ dx \\\large = \displaystyle\frac{sin^{n+1}x \ cos^{m-1} x}{n + m}+\frac{m-1}{n + m}\displaystyle \int sin^n x \ cos^{m-2} x \ dx + C$

Exponential and Logarithmic Integrals

  • $\displaystyle\large\int x^{n} \ e^{ax} \ dx = \frac{1}{a}x^{n}e^{ax} - \frac{n}{a}\displaystyle\int x^{n-1} \ e^{ax} \ dx + C $
  • $\displaystyle\large\int e^{ax} \ sin \ bx \ dx = \frac{e^{ax}}{a^{2} + b^{2}} \displaystyle[a \ sin \ bx - b \ cos \ bx] + C $
  • $\displaystyle \large\int e^{ax} \ cos \ bx \ dx = \frac{e^{ax}}{a^{2} + b^{2}} \displaystyle[a \ cos \ bx \ + b \ sin \ bx] + C$
  • $\displaystyle \large\int ln x \ dx = x \ ln x - x + C $
  • $\displaystyle \large\int x^{n} \ ln x \ dx = \frac{x^{n+1}}{[u + 1]^{2}}\displaystyle([n+1]ln x - 1) + C $
  • $\displaystyle \large\int \frac{1}{x \ ln x} \ dx = ln|ln x| + C$
